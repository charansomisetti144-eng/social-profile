# Frontend Mentor - Social Links Profile Solution

This is my solution to the Social Links Profile challenge on Frontend Mentor. The project focuses on building a responsive profile card with hover and focus states for interactive elements.

## Overview

### The Challenge

Users should be able to:

### What I Learned

During this project, I practiced:

* Semantic HTML structure
* Flexbox centering techniques
* Responsive design using media queries
* Styling buttons with hover and focus states
* Improving accessibility for keyboard users
* Organizing CSS for better readability

Example:

```css
body{
    min-height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
}
```

This helped me understand how to center content both horizontally and vertically using Flexbox.
